# Expense Tracker

**Автор:** [Беренгард Анна]

## Описание программы

Expense Tracker — это приложение для отслеживания личных расходов. Оно позволяет пользователям записывать свои расходы, фильтровать их по категории и дате, а также подсчитывать общую сумму за выбранный период.

## Как использовать

1. Запустите приложение.
2. Введите сумму, категорию и дату расходов.
3. Нажмите кнопку «Добавить расход», чтобы сохранить его.
4. Для просмотра общей суммы расходов нажмите «Подсчёт суммы».

## Примеры использования

- Введите сумму: `100`, категорию: `Еда`, дату: `2023-10-01`.
- Введите сумму: `50`, категорию: `Транспорт`, дату: `2023-10-02`.
- Нажмите «Подсчёт суммы», чтобы увидеть общую сумму расходов.
